# Ancient History RAG

A Retrieval-Augmented Generation (RAG) system for querying ancient history texts (Herodotus, Thucydides, Plutarch, and more). Supports BM25, dense (FAISS), and hybrid retrieval, with answers generated by Llama 3 8B via Ollama.

## Architecture

```
┌──────────────────────────────────────────────┐
│              Docker Host (Ubuntu)            │
│                                              │
│  ┌─────────────────┐   ┌──────────────────┐  │
│  │   app : 8501    │──▶│  ollama : 11434  │  │
│  │   (Streamlit)   │   │  (Llama 3 8B)    │  │
│  └─────────────────┘   └──────────────────┘  │
│                                              │
└──────────────────────────────────────────────┘
```

| Container | Role |
|-----------|------|
| `app` | Streamlit frontend + FAISS/BM25 retrieval |
| `ollama` | Llama 3 8B inference server |
| `ollama-init` | One-shot container that pulls the `llama3` model on first run |

## Requirements

- Ubuntu 20.04+
- Docker Engine 24+ and Docker Compose v2
- At least **32 GB RAM**, **20 GB free disk space**

## Deployment

### 1. Install Docker

```bash
curl -fsSL https://get.docker.com | sh
sudo usermod -aG docker $USER
newgrp docker
```

### 2. Clone the repository

```bash
git clone https://github.com/Yang12345-hub/6120-final-project.git
cd 6120-final-project
```

### 3. Start all services

```bash
docker compose up -d
```

This will:
1. Start the Ollama server
2. Automatically download the Llama 3 model (~4.7 GB, **first run only**)
3. Build and start the Streamlit app

> First run takes 10–20 minutes due to the model download. Subsequent starts are instant since the model is cached in a Docker volume.

### 4. Monitor startup progress

```bash
# Watch the model download
docker compose logs -f ollama-init
```

Wait until you see:
```
ancient-rag-ollama-init  | llama3 model ready
```

### 5. Open the app

```
http://<your-server-ip>:8501
```

If you're on a cloud VM (e.g. GCE), make sure port 8501 is open in the firewall:

```bash
# UFW
sudo ufw allow 8501

# GCP
gcloud compute firewall-rules create allow-streamlit \
  --allow tcp:8501 --target-tags=http-server
```

## Common Commands

```bash
docker compose up -d --build app
```

## Troubleshooting

**App returns no answers / connection error to Ollama**
- Model may still be downloading: `docker compose logs -f ollama-init`
- Check Ollama is healthy: `docker compose ps` → `ollama` should show `(healthy)`

**`ollama` shows `(unhealthy)` after startup**
- Give it another 30 seconds — healthcheck uses `ollama list` which can be slow on first boot
- Check logs: `docker compose logs ollama`

**Out of memory / model fails to load**
- Llama 3 8B needs ~8 GB RAM for the model alone. Ensure at least 32 GB total RAM.

**Port 8501 not accessible from outside**
- Check firewall rules on your cloud provider or `sudo ufw status`
